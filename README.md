This texture pack is still in a very early stage of development. Currently, only selected textures from Tech Reborn have been reworked and brought up to the visual standard of the Faithful x64 style. The goal is to consistently transfer the aesthetics of the original Faithful project to the content of the Tech Reborn mod without losing its characteristic design features.

## Download

### Prerequisites

- Install [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-10.0.0-windows-x64-installer) from Microsoft.

### Build

- Open the green dropdown button ("<> Code") on this page.
- Click on "Download ZIP" and extract it.
- On Windows, run the "build-pack.bat" file to build this pack.
- The resulting zip file is created in the same directory and can be placed into Minecraft.
